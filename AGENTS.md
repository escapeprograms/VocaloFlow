
## DataSynthesizer Pipeline

The DataSynthesizer uses a target-first pipeline with iterative alignment:

- **Entry points:** `pipelines/synthesize_v2.py` (single song), `pipelines/synthesize_dataset_v2.py` (dataset)
- **Alignment:** `alignment/iterative_align.py` — adjusts note durations before generation (no audio warping)

The legacy v1 pipeline (MFA + segmented DTW) has been fully removed.
